# Filas-em-Java

Projeto desenvolvido para estudar **filas em Java**, com foco principal em **PriorityQueue**, estrutura muito utilizada quando elementos precisam ser atendidos por prioridade.

## 📚 Conteúdo do Projeto

Este repositório contém exemplos práticos de:

- ✅ `PriorityQueue` com números inteiros
- ✅ Ordenação crescente (padrão)
- ✅ Ordenação decrescente (`reverseOrder`)
- ✅ Uso de `Comparable`
- ✅ Fila de chamados técnicos
- ✅ Fila de pacientes por prioridade
